Seja Bem-vindo ao Curso de Desenvolvimento de Software Web!

Nesta página, você encontrará os principais conceitos sobre HTML, CSS e JavaScript.

Introdução ao HTML

O HTML é a base de toda página web. Ele define a estrutura e o conteúdo de um site.

Com o HTML, você cria textos, imagens, links e diversos outros elementos visuais e semânticos.

Vamos Praticar?

Crie um arquivo index.html e adicione uma estrutura básica com título e parágrafos.

Experimente usar tags como <header>, <main> e <footer>.

Introdução ao CSS

O CSS é responsável pela aparência das páginas. Ele define cores, tamanhos, margens e muito mais.

Com o CSS, é possível transformar um simples HTML em um site visualmente atraente.

Vamos Praticar?

Crie um arquivo styles.css e conecte-o ao seu HTML com a tag <link>.

Experimente mudar a cor do fundo e do texto, adicionar margens e definir fontes diferentes.

Introdução ao JavaScript

O JavaScript adiciona interatividade às páginas, permitindo respostas dinâmicas do usuário.

É usado para validar formulários, criar animações e desenvolver aplicações web completas.

Vamos Praticar?

Crie um arquivo script.js e use alert() para exibir uma mensagem.

Depois, explore como capturar eventos de clique e alterar o conteúdo da página.
